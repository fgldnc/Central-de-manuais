# Central de Manuais — Departamento Financeiro GBA

Central de consulta dos procedimentos operacionais do setor financeiro do escritório GBA Advogados.

## Publicação no GitHub Pages

1. Crie um repositório no GitHub (ex: `central-manuais-gba`)
2. Faça upload do arquivo `index.html`
3. Acesse **Settings → Pages**
4. Em **Source**, selecione a branch `main` e a pasta `/ (root)`
5. Clique em **Save**
6. O link será gerado automaticamente (ex: `https://seunome.github.io/central-manuais-gba`)

## Como atualizar a central

Para atualizar um manual existente ou adicionar um novo, envie o arquivo Word atualizado neste projeto do Claude junto com a instrução abaixo:

```
Atualize apenas o manual [NOME DO MANUAL], na seção [NOME DA SEÇÃO].

Trecho antigo:
[cole o trecho que vai sair]

Trecho novo:
[cole o trecho que entra]

Regras:
- Não reescreva a central inteira
- Preserve estrutura, sidebar, busca e identidade visual GBA
- Entregue apenas o arquivo alterado
- Informe o que mudou em 1 linha
- Atualize a data de última atualização do manual
```

Após receber o `index.html` atualizado, substitua o arquivo no repositório e o GitHub Pages atualizará automaticamente.

## Conteúdo

A central contém 24 manuais organizados nas seguintes categorias:

- Conciliação e Controle
- Notas Fiscais e Fiscal
- Cobranças e Inadimplência
- Folha e Pagamentos
- Comissionamento
- Relatórios e DRE
